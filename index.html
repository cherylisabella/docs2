vangogh

Van Gogh-inspired colour palettes for R with accessibility, metadata, exports, and ggplot2 integration. Transform your data visualizations with the artistic beauty of Vincent van Gogh's masterpieces.

ggplot2 visualization accessibility color palettes
install.packages("vangogh")

Key Features

  • 15+ painterly palettes from iconic Van Gogh artworks
  • ggplot2-compatible color scales and custom themes
  • Comprehensive accessibility checks for color blindness
  • Export palettes to JSON, CSV with metadata
  • Palette comparison and suggestion tools
  • HCL color space analysis

leakr

Comprehensive R package for detecting and diagnosing data leakage in machine learning workflows. Ensure model integrity and reliability with automated, registry-based detection systems.

machine learning data validation tidymodels mlr3
install.packages("leakr")

Key Features

  • Automated detection of train/test contamination
  • Target leakage identification with correlation analysis
  • Temporal leakage detection for time-series data
  • Duplicate and near-duplicate record detection
  • Integration with tidymodels, caret, and mlr3
  • Rich visualization and reporting capabilities
  • Data snapshots for reproducibility
  • Custom detector registration system
